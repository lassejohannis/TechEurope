# Side Challenges — Tech Europe Hack

Side prizes are **stackable on top of** the Qontext track prize. Judging per challenge requires using the partner's product in the project + confirming use in the submission.

| Challenge | Requirement | Prize |
|---|---|---|
| **Fastino — Best use of Pioneer** | Use Pioneer. Fine-tune a model that outperforms / replaces a general-purpose LLM API call. Thoughtful use of features: synthetic data generation, evaluation against frontier models, adaptive inference. **Bonus:** creative use of GLiNER2. | **700€** (Mac Mini cash value) |
| **Aikido — Most Secure Build** | Create Aikido account, connect Git, connect repo, screenshot of security report (counts + categories of issues). | **1000€** |
| **Gradium — Best use of Gradium** | Use Gradium. Confirm in submission. | 900k Gradium credits + goodies |
| **Entire — Best use of Entire** | Use Entire. Confirm in submission. | $1000 Apple gift cards + Switch 2 + PS5 + Xbox |

## Stacking strategy vs. the Qontext track

- **Entire** — already in our recommended core 3. Side challenge = essentially free add-on, just wire up the HITL flow well.
- **Aikido** — not eligible as one of the 3, but **easy 1000€**: create account, connect repo, screenshot. Do on Day 1, ignore during dev, screenshot at submission.
- **Pioneer / Fastino** — has a real fit for the Qontext case: fine-tune a small entity/relation extractor on synthetic data generated from the sample dataset → outperforms a generic Gemini call at that specific task. Also checks the "models that train themselves" angle. Strong stack-on candidate.
- **Gradium** — fit is weaker (voice doesn't naturally serve a context base). Only if we have clear slack and want the extra prize pool.

**Prize stack if we hit all of them:** Qontext track (1g gold/member + dinner) + Fastino (700€) + Aikido (1000€) + Entire ($1k gift cards + consoles) + Gradium (credits/goodies) + Stage-2 finalist prizes.
