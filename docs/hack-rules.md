# Tech Europe Hack — Rules & Submission Requirements

## Submission requirements

- **Deadline:** Sunday 14:00
- **Team size:** max. 5 people
- **Partner tech:** use **min. 3 partner technologies** (listed under resources)
- **Newly built at the hackathon** (boilerplates allowed)

### What to submit

**Project presentation**
- 2-minute video demo (Loom or equivalent)
- Must include:
  - Detailed explanation of the solution
  - Live walkthrough of key features

**Open-source repository**
- Public GitHub repo with the source code
- Must include:
  - README with setup + installation instructions
  - Clear documentation of all APIs, frameworks, tools used
  - Technical documentation sufficient for jury evaluation

## Competition mode — two stages

### Stage 1: Pre-Selection
- Complete freedom of topic within the chosen track
- **8 finalist teams (1 per track)** advance
- Judging: creativity + technical complexity + bonus for effective use of partner tech

### Stage 2: Finalist Stage
- Live presentation to jury + audience
- **5-minute pitch** per team
- Jury picks **top 3 winners** → Finalist Stage Prizes
